classicborne is a general-purpose Node.js library for building Minecraft Classic servers. It provides a set of extensible base classes — `BaseUniverse`, `BaseLevel`, and `BasePlayer` — that you subclass to add your own game logic, level generation, player authentication, and persistence.

## What classicborne provides

classicborne handles the low-level Minecraft Classic protocol so you can focus on building your server's unique features.

<CardGroup cols={2}>
  <Card title="CPE support" icon="plug">
    Full support for Classic Protocol Extensions — custom blocks, block definitions, player lists, environment settings, and more.
  </Card>
  <Card title="Extensible classes" icon="layer-group">
    Every core concept is a class you can subclass. Override methods to add custom logic without touching protocol internals.
  </Card>
  <Card title="Multi-level management" icon="map">
    Load, manage, and unload multiple worlds within a single server instance. Players can move between levels dynamically.
  </Card>
  <Card title="Block change persistence" icon="floppy-disk">
    An append-only change record stores every block placement with optional keyframe snapshots for fast restoration.
  </Card>
</CardGroup>

## Minimal server

```js quickstart.mjs
import { BaseUniverse } from "classicborne/class/server/BaseUniverse.mjs"

const universe = new BaseUniverse({
  port: 25565,
  postToMainServer: false,
})
```

<Note>
  Always import classes directly from their module paths (e.g. `classicborne/class/server/BaseUniverse.mjs`) rather than from the barrel `classicborne` index. The barrel file is intended only for documentation generation and some imports may fail at runtime.
</Note>
